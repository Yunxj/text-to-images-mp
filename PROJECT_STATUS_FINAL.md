### 2. 项目导入
```bash

```
1. 打开微信开发者工具
2. 选择"导入项目"
3. 输入上述路径和你的AppID
4. 点击"导入"

### 3. 配置修改
更新以下文件中的配置：

- `utils/api.js`: 配置生产环境API地址

### 4. 发布流程
1. **本地测试** → 在开发者工具中测试所有功能
2. **真机预览** → 扫码在手机上测试
3. **上传代码** → 点击"上传"按钮
4. **提交审核** → 在微信公众平台提交审核
5. **发布上线** → 审核通过后点击"发布"
